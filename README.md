# 🪨 📄 ✂️ Rock, Paper, Scissors Game

A classic, interactive Rock, Paper, Scissors game built from scratch using frontend web technologies. Play against the computer, track your score, and see who dominates!


## ✨ Features
* **Interactive UI:** Clean, modern, and responsive design that works on mobile and desktop.
* **Smart Scoring:** Tracks the user's and computer's scores in real-time during the session.
* **Instant Feedback:** Visually shows who won, lost, or tied each round with dynamic messages and animations.
* **Reset Button:** Easily restart the game and wipe the scores clean whenever you want.

---

## 🛠️ Tech Stack Used
* **HTML5:** Structure of the game elements.
* **CSS3:** Custom styling, layout (Flexbox/Grid), and smooth transitions.
* **JavaScript (ES6+):** Game logic, event listeners, computer choice generation, and DOM manipulation.
